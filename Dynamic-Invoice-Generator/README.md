# jquery-invoice-plugin
jQuery Plugin for editable and dynamic Invoice with calculation.

# Requirement
<a href="https://jquery.com/">jQuery</a> version >= 1.12.4

# Usage
    jQuery().invoice({
      addRow : "#addRow",
      delete : ".delete",
      parentClass : ".item-row",

      price : ".price",
      qty : ".qty",
      total : ".total",
      totalQty: "#totalQty",

      subtotal : "#subtotal",
      discount: "#discount",
      shipping : "#shipping",
      grandTotal : "#grandTotal"
    });
